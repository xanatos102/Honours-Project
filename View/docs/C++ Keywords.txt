C++ Key Words


break 		- exits from for and while loops
case  		- used with switch statements
char  		- character data type
default		- used with switch statement
do		- used in do-while loop
double		- real number data type
else		- used in if-else statements
float		- real number data type
for		- used in for-loops
if		- used in if-else statements
int		- whole number data type
long		- whole number data type
return		- returns from function
short		- whole number data type
signed		- data type modifier
sizeof		- returns the size of a specified data type
struct		- used to define user data types
switch		- used in switch statement
unsigned	- data type modifier
void		- function which does not return any data types
while		- used in while or do-while statement
